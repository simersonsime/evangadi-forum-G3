# Team Task Assignment

This task assignment shows how backend and frontend tasks are shared among team members. Each student is responsible for one backend task and one frontend task.

The project uses one shared GitHub repository. All team members must work in the same repository, contribute every day, and push their code to GitHub daily. Clear commit messages and teamwork are required for smooth project development.
