Frontend Commands Reference - for Evangadi Forum project
1. Project Initialization (Vite + React)
- npm create vite@latest frontend - Create a new Vite + React project
- cd frontend - Navigate into frontend folder
- npm install - Install all dependencies from package.json

2. Install Core Packages
- npm install react-router-dom - Client-side routing
- npm install axios - HTTP requests to backend APIs
- npm install jwt-decode - Decode JWT token on frontend 
- npm install dotenv - Environment variable support

3. Development & Build Commands
- npm run dev - Start development server
- npm run build - Build production-ready frontend
- npm run preview - Preview production build locally

4. Common npm Commands
- npm install           # Install dependencies
- npm uninstall <pkg>   # Remove a package
- npm update            # Update packages
- npm list --depth=0    # List installed packages
- npm audit             # Check vulnerabilities
- npm audit fix         # Fix vulnerabilities

5. Environment Variables
- Create .env file
- Add API base URL: VITE_API_BASE_URL=http://localhost:4000/api

6. Frontend Folder Structure
frontend/
├── public/ # Static assets
├── src/
│ ├── assets/       # Images, icons
│ ├── components/   # Header, Footer, ProtectedRoute
│ ├── pages/        # Home, Login, Signup, Question, Answer, AskQuestion
│ ├── services/     # api.js (Axios config)
│ ├── context/      # AuthContext.jsx
│ ├── utils/        # auth helpers
│ ├── App.jsx       # Routes setup
│ ├── main.jsx      # App entry point
│ └── index.css     # Global styles
│
├── .env            # Frontend env variables
├── vite.config.js
├── package.json
└── package-lock.json

7. Workflow Summary
  1. Create frontend → npm create vite@latest
  2. Install dependencies → npm install
  3. Install routing & axios → npm install react-router-dom axios
  4. Setup .env with backend API URL
  5. Configure Axios instance
  6. Implement AuthContext & ProtectedRoute
  7. Connect frontend to backend APIs
  8. Run app → npm run dev
